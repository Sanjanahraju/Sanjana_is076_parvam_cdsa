# Personal Expense Tracker

A simple web application built with Python and Flask to track personal expenses.

## Features

- Log daily expenses with item name, amount, and category
- Categories: Food, Transport, Shopping, Other
- Automatic recalculation of category totals and grand total
- Warning icon for expenses exceeding ₹500
- Monthly budget summary (shows all expenses and totals)

## Installation

1. Install Python 3.x
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open http://127.0.0.1:5000/ in your browser

## Usage

- Add expenses using the form
- View all expenses in the table
- Delete expenses with the delete button
- Totals update automatically
